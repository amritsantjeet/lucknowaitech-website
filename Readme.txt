DESCRIPTION
-----------
This is a professional, modern company website for "Lucknow Tech Labs" - 
an IT, Software Development and AI consultancy company based in Lucknow.

It is built using Python + Flask framework. No frontend framework 
(like React) is used - everything is simple HTML, CSS (Bootstrap) and Python.

FEATURES
--------
- Beautiful homepage with gradient hero section
- Services section (Custom Software, AI/ML, 24/7 Support)
- Flagship Products section (Student Manager, Billing Manager)
- Why Choose Us section with icons
- Responsive design (works on mobile and desktop)
- Custom domain connected: lucknowtechlabs.com
- Hosted live on Render.com (free)

TECHNOLOGIES USED
-----------------
- Python 3
- Flask (Web Framework)
- Bootstrap 5 (for beautiful design)
- Gunicorn (production server)
- Render.com (free hosting)
- Git + GitHub

FOLDER STRUCTURE
----------------
lucknowtechlabs/
├── app.py                 ← Main Python file
├── requirements.txt       ← List of packages
├── Procfile               ← For Render deployment
├── templates/
│   └── index.html         ← Main webpage
└── README.txt             ← This file

HOW TO RUN LOCALLY
------------------
1. Open the project folder in VS Code
2. Open terminal and run:
   pip install -r requirements.txt
3. Run the website:
   python app.py
4. Open browser and go to: http://127.0.0.1:5000

HOW TO DEPLOY / UPDATE
----------------------
- Code is connected to GitHub
- Any change you push to GitHub automatically updates the live website
- Live website: https://lucknowtechlabs.onrender.com
- Custom domain: https://lucknowtechlabs.com
